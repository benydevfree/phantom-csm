## Codebase Patterns
- DB client : `import { db } from './db'` — `db.query(sql, params)`, `db.end()` at end of migrate script
- migrate.ts uses sequential `await db.query(...)` calls, each followed by `console.log('✅ ...')`
- UUID primary keys : `UUID PRIMARY KEY DEFAULT gen_random_uuid()`
- Conditional index creation : `CREATE UNIQUE INDEX IF NOT EXISTS ... WHERE col IS NOT NULL`
- No ORM — raw SQL only
